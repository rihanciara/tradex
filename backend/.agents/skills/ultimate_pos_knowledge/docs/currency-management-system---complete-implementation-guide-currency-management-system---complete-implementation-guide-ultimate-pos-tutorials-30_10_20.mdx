# Currency Management System for Ultimate POS

This comprehensive guide covers the complete implementation of a Currency Management system in Ultimate POS with Laravel, including CRUD operations, DataTables integration, Tailwind CSS styling, and proper handling of currency formatting options.

## Overview​

The Currency Management system provides:

  * ✅ Complete CRUD operations for currencies
  * ✅ DataTables with server-side processing
  * ✅ Modal-based create, edit, and view operations
  * ✅ Tailwind CSS styling for modern UI
  * ✅ Proper handling of thousand and decimal separators
  * ✅ Support for space as thousand separator
  * ✅ Permission-based access control
  * ✅ Transaction integration checking
  * ✅ Multi-language support with translation keys

## Model Implementation​

Create the 
```text
Currency
```
 model at 
```text
app/Currency.php
```
:


```php
<?php

namespace App;

use Illuminate\Database\Eloquent\Model;

class Currency extends Model
{
    /**
     * Indicates if the model should be timestamped.
     *
     * @var bool
     */
    public $timestamps = true;

    /**
     * The attributes that are mass assignable.
     *
     * @var array
     */
    protected $fillable = [
        'country',
        'currency',
        'code',
        'symbol',
        'thousand_separator',
        'decimal_separator',
    ];

    /**
     * Get all transactions for this currency.
     */
    public function transactions()
    {
        return $this->hasMany(\App\Transaction::class);
    }
}
```

### Index Method (DataTables)​


```php
public function index()
{
    if (!auth()->user()->can('currency.view')) {
        abort(403, 'Unauthorized action.');
    }

    if (request()->ajax()) {
        $currencies = Currency::select([
            'country',
            'currency',
            'code',
            'symbol',
            'thousand_separator',
            'decimal_separator',
            'id'
        ]);

        return datatables()->of($currencies)
            ->addColumn('action', function ($row) {
                $html = '<div class="btn-group">';
                if (auth()->user()->can('currency.view')) {
                    $html .= '<button class="btn btn-info btn-xs" title="View" onclick="viewCurrency(' . $row->id . ')"><i class="fa fa-eye"></i></button>';
                }
                if (auth()->user()->can('currency.edit')) {
                    $html .= '<button class="btn btn-warning btn-xs edit_currency_button" title="Edit" data-href="' . action([\App\Http\Controllers\CurrencyController::class, 'edit'], [$row->id]) . '"><i class="fa fa-edit"></i></button>';
                }
                if (auth()->user()->can('currency.delete')) {
                    $html .= '<button class="btn btn-danger btn-xs delete_currency_button" title="Delete" data-href="' . action([\App\Http\Controllers\CurrencyController::class, 'destroy'], [$row->id]) . '"><i class="fa fa-trash"></i></button>';
                }
                $html .= '</div>';
                return $html;
            })
            ->removeColumn('id')
            ->rawColumns([6])
            ->make(false);
    }

    return view('currency.index');
}
```


### Store Method (Create)​


```php
public function store(Request $request)
{
    if (!auth()->user()->can('currency.create')) {
        abort(403, 'Unauthorized action.');
    }

    $validatedData = $request->validate([
        'country' => 'required|max:255',
        'currency' => 'required|max:255',
        'code' => 'required|max:10|unique:currencies,code',
        'symbol' => 'required|max:20',
        'thousand_separator' => 'nullable|max:5',
        'decimal_separator' => 'required|max:5',
    ]);

    try {
        // Set default value if thousand_separator is not provided, null, or empty string
        // But allow spaces and other characters to be preserved
        if (!isset($validatedData['thousand_separator']) ||
            $validatedData['thousand_separator'] === null ||
            $validatedData['thousand_separator'] === '') {
            $validatedData['thousand_separator'] = ',';
        }

        Currency::create($validatedData);

        if ($request->ajax()) {
            return response()->json([
                'success' => true,
                'msg' => __('lang_v1.currency_added_success'),
            ]);
        }

        return redirect()->route('currencies.index')
                        ->with('success', __('lang_v1.currency_added_success'));
    } catch (\Exception $e) {
        if ($request->ajax()) {
            return response()->json([
                'success' => false,
                'msg' => __('messages.something_went_wrong'),
            ]);
        }

        return redirect()->route('currencies.index')
                        ->with('error', __('messages.something_went_wrong'));
    }
}
```

public function update(Request $request, Currency $currency)
{
    if (!auth()->user()->can('currency.edit')) {
        abort(403, 'Unauthorized action.');
    }

    $validatedData = $request->validate([
        'country' => 'required|max:255',
        'currency' => 'required|max:255',
        'code' => 'required|max:10|unique:currencies,code,' . $currency->id,
        'symbol' => 'required|max:20',
        'thousand_separator' => 'nullable|max:5',
        'decimal_separator' => 'required|max:5',
    ]);

    try {
        // Set default value if thousand_separator is not provided, null, or empty string
        // But allow spaces and other characters to be preserved
        if (!isset($validatedData['thousand_separator']) ||
            $validatedData['thousand_separator'] === null ||
            $validatedData['thousand_separator'] === '') {
            $validatedData['thousand_separator'] = ',';
        }

        $currency->update($validatedData);

        if ($request->ajax()) {
            return response()->json([
                'success' => true,
                'msg' => __('lang_v1.currency_updated_success'),
            ]);
        }

        return redirect()->route('currencies.index')
                        ->with('success', __('lang_v1.currency_updated_success'));
    } catch (\Exception $e) {
        if ($request->ajax()) {
            return response()->json([

                'msg' => __('messages.something_went_wrong'),
            ]);
        }

        return redirect()->route('currencies.index')
                        ->with('error', __('messages.something_went_wrong'));

}

### Destroy Method (Delete)​


```php
public function destroy(Currency $currency)
{
    if (!auth()->user()->can('currency.delete')) {
        abort(403, 'Unauthorized action.');
    }

    // Check if currency is being used in transactions
    $transactionCount = DB::table('transactions')->where('currency_id', $currency->id)->count();

    if ($transactionCount > 0) {
        if (request()->ajax()) {
            return response()->json([
                'success' => false,
                'msg' => __('lang_v1.currency_in_use'),
            ]);
        }

        return redirect()->route('currencies.index')
                        ->with('error', __('lang_v1.currency_in_use'));
    }

    try {
        $currency->delete();

        if (request()->ajax()) {
            return response()->json([
                'success' => true,
                'msg' => __('lang_v1.currency_deleted_success'),
            ]);
        }

        return redirect()->route('currencies.index')
                        ->with('success', __('lang_v1.currency_deleted_success'));
    } catch (\Exception $e) {
        if (request()->ajax()) {
            return response()->json([
                'success' => false,
                'msg' => __('messages.something_went_wrong'),
            ]);
        }

        return redirect()->route('currencies.index')
                        ->with('error', __('messages.something_went_wrong'));
    }
}
```

## Middleware Configuration​

### TrimStrings Middleware​

This is **CRITICAL** for preserving spaces in thousand separators. Update 
```text
app/Http/Middleware/TrimStrings.php
```
:


```text
<?php

namespace App\Http\Middleware;

use Illuminate\Foundation\Http\Middleware\TrimStrings as BaseTrimmer;

class TrimStrings extends BaseTrimmer
{
    /**
     * The names of the attributes that should not be trimmed.
     *
     * @var array
     */
    protected $except = [
        'password',
        'password_confirmation',
        'prefix', //Invoice scheme prefix
        'ref_no_prefixes.purchase', //Business settings prefixes
        'ref_no_prefixes.purchase_return',
        'ref_no_prefixes.stock_transfer',
        'ref_no_prefixes.stock_adjustment',
        'ref_no_prefixes.sell_return',
        'ref_no_prefixes.expense',
        'ref_no_prefixes.contacts',
        'ref_no_prefixes.purchase_payment',
        'ref_no_prefixes.sell_payment',
        'ref_no_prefixes.expense_payment',
        'ref_no_prefixes.business_location',
        'ref_no_prefixes.username',
        'ref_no_prefixes.subscription',
        'ref_no_prefixes.draft',
        'thousand_separator', //Don't trim thousand separator (can be space)
        'decimal_separator', //Don't trim decimal separator (can be space or other chars)
    ];
}
```


### Why This Matters​

Laravel's 
```text
TrimStrings
```
 middleware automatically removes leading and trailing whitespace from all request inputs. Without excluding 
```text
thousand_separator
```
 and 
```text
decimal_separator
```
, spaces would be stripped, preventing users from using space as a separator.

## Troubleshooting​

### Issue: "Processing..." message in DataTable​

**Cause** : Column count mismatch between server response and table headers

**Solution** : Ensure 
```text
rawColumns([6])
```
 matches 
```text
targets: [6]
```
 and column count is correct

### Issue: Timestamps show "N/A"​

**Cause** : Model missing 
```text
$timestamps = true
```


**Solution** : Add 
```text
public $timestamps = true;
```
 to Currency model

### Issue: Space separator not saving​

**Cause** : 
```text
TrimStrings
```
 middleware removing spaces

**Solution** : Add 
```text
thousand_separator
```
 and 
```text
decimal_separator
```
 to middleware exclusion list

### Issue: Validation errors not showing​

**Cause** : Missing error callback in AJAX

**Solution** : Add error handler to show validation messages

### Issue: Action buttons not visible​

**Cause** : Column index mismatch

**Solution** : Verify column indexes match between controller and JavaScript

### Issue: Delete button visible for currency in use​

**Cause** : Check for transactions before allowing delete

**Solution** : Verify the destroy method checks transaction count
